# Seguridad

VigiScan es una herramienta defensiva. Usala solo con autorizacion explicita.

Reglas:

- No explotar vulnerabilidades.
- No enviar payloads ofensivos.
- No hacer fuerza bruta agresiva.
- No exfiltrar secretos.
- Enmascarar evidencia sensible.
- Tratar resultados como senales de triage, no como prueba absoluta.

Limitacion: VigiScan ayuda a detectar muchos riesgos web, pero ninguna
herramienta garantiza detectar el 100% de vulnerabilidades.
